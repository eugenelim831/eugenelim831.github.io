Optimizing Clinical Laboratory Operations with Machine Learning

A thesis exploring how ML can improve blood testing workflows (e.g., predicting haemolysis risk and operational delays) with transparent, ethical practice.

Overview

This project studies the relationship between blood specimen potassium (K) levels, haemolysis, and logistics (e.g., travel time from GP to lab), aiming to inform lab operations and reduce avoidable redraws. Models include OLS, Random Forest, XGBoost, and MLP, with SHAP/LIME/Grad-CAM used to support interpretability.

Objectives

  • Model haemolysis and related risks to support lab decisions.
  • Quantify impact of transit and processing times.
  • Adopt human-centred, transparent ML practices.

Outcomes

  • Benchmark of classic vs. ensemble models.
  • Feature insights via SHAP & permutation importance.
  • Ethics review using ALTAI & Microsoft AI Fairness Checklist.
